## 项目笔记结构

每个项目笔记包含：

1. **项目概述**：一句话描述、技术栈
2. **架构图**：系统架构、核心流程
3. **我的职责**：具体负责哪些模块
4. **技术亮点**：2-3个亮点，有数据支撑
5. **遇到的问题**：问题→方案→结果
6. **面试题准备**：这个项目会被问什么
