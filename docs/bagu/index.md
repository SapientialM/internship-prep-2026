## 学习路径建议

### 阶段一：核心基础（优先级最高）

**JavaSE → JUC → JVM → MySQL → Redis → Spring**

这些是后端面试的绝对核心，每轮面试都会涉及。建议按顺序学习，因为：
- JavaSE 是语法基础，动态代理、反射、集合源码是高频考点
- JUC 是并发基础，线程池、锁、ThreadLocal 几乎必问
- JVM 是性能基础，内存模型、GC、调优是区分度的关键
- MySQL + Redis 是数据基础，索引、事务、缓存策略必考

### 阶段二：系统设计

**分布式 → 场景题 → 计算机网络 → 操作系统**

- 分布式和场景题通常在大厂二面/三面出现
- 计算机网络和操作系统作为基础补充

### 阶段三：专项突破

**Kafka → 分库分表 → 问题排查 → AI大模型**

- 根据简历中的技术栈选择性深入
- AI大模型是当前热点，建议了解 RAG、Function Calling、MCP 等概念
