## 为什么要手写？

1. **理解底层原理** — 看到源码≠理解源码，亲手实现才能真正理解设计权衡
2. **面试加分项** — 讲动态代理/集合源码时能结合手写经验，更有说服力
3. **提升代码能力** — 手写关键组件需要处理边界条件、异常、性能，锻炼工程能力
