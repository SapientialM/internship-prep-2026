## 🚀 快速开始

### 在线访问

🌐 **博客地址**: https://sapientialm.github.io/internship-prep-2026

### 本地运行

```bash
# 1. 克隆仓库
git clone git@github.com:SapientialM/internship-prep-2026.git
cd internship-prep-2026

# 2. 安装依赖
npm install

# 3. 本地预览
npm run server

# 4. 访问 http://localhost:4000
```

### 常用命令

```bash
# 本地预览
npm run server

# 创建新文章
node new-post.js tech "文章标题"

# 构建
npm run build

# 部署
npm run deploy
```
