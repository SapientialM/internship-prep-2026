# 16 - 智力题

> 智力题在大厂面试中偶尔出现，主要考察逻辑思维和分析问题的能力。

---

## 1. 经典智力题

### 赛马问题

**题目：** 25 匹马，每次最多 5 匹比赛，找出前 3 名，最少需要几场比赛？

**答案：** 7 场。

**思路：**

1. 分 5 组，每组 5 匹比赛（5 场）
2. 每组第一名比赛，决出最快的一匹（1 场）
3. 最快马所在组的前 3 名 + 第二名所在组的前 2 名 + 第三名所在组的第 1 名（共 5 匹）再比一次（1 场）
4. 第 7 场比赛的前 3 名即最终结果

### 天平称重

**题目：** 12 个球，其中 1 个重量不同（不知轻重），用天平称 3 次找出。

**思路：** 分 3 组（4, 4, 4），首次称两组：
- 平衡 → 异常球在第 3 组，用标准球对比
- 不平衡 → 异常球在 8 个中，根据天平倾斜判断轻重

### 扔鸡蛋

**题目：** 2 个鸡蛋，100 层楼，找到鸡蛋不碎的最高楼层，最少需要几次？

**答案：** 14 次。

**思路：** 第一个鸡蛋决定区间，第二个鸡蛋在区间内线性尝试。最优策略是让每多扔一次鸡蛋 1，鸡蛋 2 的尝试次数少 1 次。所以第一个间隔从 14 开始，依次 13、12...，和为 105 > 100。即：14 → 27 → 39 → 50 → 60 → 69 → 77 → 84 → 90 → 95 → 99 → 100。

### 烧绳子

**题目：** 两根不均匀的绳子，每根烧完 1 小时，如何测量 45 分钟？

**答案：** 第一根两头同时点（30 分钟烧完），同时第二根点一头。第一根烧完时（30 分钟已过）立即将第二根的另一头也点燃，第二根剩余部分将在 15 分钟内烧完。30 + 15 = 45 分钟。

## 2. 概率题

### 囚徒开关问题

**题目：** 100 个囚徒，1 个房间有电灯（初始关）。每天随机 1 个囚徒进房间，可以选择开/关灯。只要有 1 个囚徒能说出"所有囚徒都进过房间至少一次"，所有人释放。商量策略。

**思路：** 指定 1 人计数，其他人每人第一次进房间且灯关着时开灯（只开一次）。计数的人每次进房间灯开着就计数+1 并关灯。计数到 99 即知所有人都进过。

### 生男生女概率

**题目：** 一个国家所有家庭都生育直到生出男孩为止。长期后男女比例？

**答案：** 1:1。因为每次生育是独立事件，男/女概率始终是 1/2。家庭生育策略不影响整体的期望比例。

## 3. 逻辑推理

### 帽子颜色

**题目：** 3 个人，帽子颜色可能是红或蓝（至少 1 红）。每个人能看到其他两人帽子但看不到自己的。依次回答自己帽子的颜色。

**思路：**
- 第 1 人回答不出 → 他看到至少 1 顶红帽
- 第 2 人回答不出 → 他看到至少 1 顶红帽（不含第 1 人看到的可能）
- 第 3 人推断：如果他的帽子是蓝的，第 2 人会推断出自己是红的（因为第 1 人看到 1 顶红帽）→ 第 2 人没回答，说明第 3 人是红的。

## 4. 估算题

### 北京有多少个加油站？

**思路：**
1. 北京面积 ≈ 16410 km²
2. 假设城区占 20%，即 3282 km²
3. 加油站服务半径约 2km²一个
4. 城区加油站 ≈ 3282 / 2 ≈ 1641 座
5. 加上郊区，总共约 2000-2500 座

**关键：不是追求精确结果，而是展现合理的估算思路。**
